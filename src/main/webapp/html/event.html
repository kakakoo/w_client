2019년 설 연휴
설 연휴 기간 환전
사전 예약 안내

2019년 설 연휴 기간 환전 고객님들을 위해 '사전 예약'을 진행합니다.

사전 예약 접수 기간 외에는 설 연휴 기간 수령일 지정이 불가하오니 미리 확인하신 후 이용 부탁드립니다.

1. 설 연휴 기간 환전 사전 예약 안내

- 예약 접수기간: 2019/01/25(금) 11시 00분 ~ 2019/01/31(목) 23시 59분
- 수령가능 기간: 2019/02/02(토) 05시 00분 ~ 2019/02/07(목) 21시 00분
(수령 가능 기간 매일 오전 5시부터 오후 9시까지 정상 수령 가능)

→ 웨이즈 홈페이지 또는 모바일 앱을 통해 사전 예약 가능
→ 인천국제공항 제 1터미널 수령 가능(연휴 기간 주소지 배송 불가)
→ 예약 접수기간 외 추가 예약 및 재 예약 불가
→ 예약 후 30분 이내 입금 완료 조건, 시간 초과 시 자동 취소
→ 해당 기간 소액권 제공에 제한이 있을 수 있음
→ 예약량 증가 등 서비스 사정에 의해 사전 예약 조기 마감 가능

2. 설 연휴 사전 예약 취소 위약금 안내

- 결제 후 ~ 수령예정일 1일 전까지: 없음
- 수령예정일 당일 0시 이후: 3,000원

→ 입금 완료 후 환불 요청 시 이미 사용된 쿠폰 반환 불가(재사용 불가)
→ 설 연휴 기간 접수된 환불 건은 2019년 2월 7일부터 순차적 환불 진행

3. 고객센터 운영시간 및 휴무 안내

- 전화상담: 2019/02/02(토) ~ 2019/02/06(수) 휴무
- 앱 내 채팅상담: 2019/02/02(토) ~ 2019/02/06(수) 24시간
- 카카오톡 채팅상담: 2019/02/02(토) ~ 2019/02/06(수) 24시간

→ 수령일 당일 수령 안내 및 수령 시간 변경은 배송 담당자 휴대전화를 통해서도 상담 가능(매일 오전 5시 ~ 오후 9시)
